1) Любой объект типа String умеет преобразовать строку в заглавные буквы с помощью метода string.toUpperCase(); 
2) Любой объект типа String умеет проверить пуста строка или нет, с помощью метода string.isEmpty();
3) Любой объект типа Object умеет сравнивать два объекта с помощью метода equals() и возвращает true, если они идентичны; object1.equals(object2);
